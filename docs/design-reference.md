# Design Reference

The visual direction is based on the provided Cadence design archive:

- Fixed desktop sidebar
- Soft academic SaaS layout
- White and mist panels over a calm off-white canvas
- Blue primary actions with cyan AI accents
- Rounded cards and subtle glassmorphism
- Workload forecast and study planner as the primary product surfaces

The implementation recreates the design language in Next.js and Tailwind instead of copying the exported HTML directly.
